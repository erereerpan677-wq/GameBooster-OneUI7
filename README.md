# GameBooster-OneUI7  
Custom Game Booster untuk Samsung A04e tanpa gimmick.  
Fitur:  
- Overlay FPS  
- Floating Tools  
- Optimizer ringan  
- Mode CPU Boost sederhana  
- Popup saat game berjalan  

Project ini hanya contoh UI dan struktur dasar.  
Tidak mengaktifkan fitur resmi Game Booster Samsung.  
Aplikasi ini dibuat untuk latihan (UI + Logic sederhana).
